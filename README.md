# Hide & Seek.io Prototype

難易度選択つきのHide & Seek系ioゲーム試作版です。

## 遊び方

- WASD / 矢印キー: 移動
- Space: 近くのオブジェクトに変身
- Shift: ダッシュ

## 難易度

- Easy: 鬼 3体
- Normal: 鬼 5体
- Hard: 鬼 8体
- Nightmare: 鬼 12体

難易度が上がるほど、鬼の数・移動速度・索敵感度・ソナー頻度が上がります。

## GitHub Pagesで公開する方法

1. GitHubで新しいリポジトリを作る
2. `index.html` と `README.md` をアップロード
3. Settings → Pages
4. Branch を `main`、Folder を `/root` にして保存
5. 数十秒後に公開URLが出ます
