# W:E:go Backstage Hunt Prototype

Hide & Seek系ブラウザゲーム試作版です。

## v0.3 追加要素

- 鬼AIに視野コーンを追加
- 鬼AIにスタミナゲージを追加
- 追跡・調査・巡回でスタミナ消費
- スタミナ切れで3〜6秒休憩
- 休憩中はその場でキョロキョロする
- 鬼が常にうろうろするように調整
- 壁越しに見つけにくい簡易Line of Sight判定を追加

## 遊び方

- WASD / 矢印キー: 移動
- Space: 近くのオブジェクトに変身
- Shift: ダッシュ

## 難易度

- Easy: 鬼 3体
- Normal: 鬼 5体
- Hard: 鬼 8体
- Nightmare: 鬼 12体
