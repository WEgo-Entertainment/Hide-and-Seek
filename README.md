## 操作

### PC

- WASD / 矢印キー: 移動
- Space: 近くのオブジェクトに変身
- Shift: ダッシュ
- R: 緊急リロード
- Esc: タイトル画面へ戻る

### スマホ

- 左下スティック: 移動
- 右上ボタン: ダッシュ
- 右下ボタン: 変身

## 更新方法

1. このZIPを解凍
2. 中身を `Hide-and-Seek` フォルダへ上書き
3. GitHub Desktopで `Commit`
4. `Push origin`
5. ブラウザでゲームを更新
